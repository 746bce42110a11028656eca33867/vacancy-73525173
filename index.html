База используется mysqlite. Создаётся автоматически. Просто key:value хранилице, без трюков ибо не нужны.
Создать таблицы в БД для данных чек, автоматически при любом использовании бэкэнда
Создать форму для отображения данных по всем отделениям чек здесь можно полистать
Создать функционал добавления новых данных в таблицы чек отдельная страница только для создания
Создать функционал удаления данных из таблиц чек кнопка в поле отладки
Создать функционал изменения существующих данных в таблицах чек кнопка в поле отладки
Создать функционал проверки данных при добавлении и редактировании (должны быть числа) чек, автоматически, и на фронте и на бэке, кроме времени замеров
Создать функционал экспорта данных в PDF кнопка в правом нижнем углу

Работает на PHP 8.0.6 Development Server . На более старых версиях php работа не гарантируется, а на версиях < 7 сломается обязательно.
